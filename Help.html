Базы данных

Портфели храняться в базе данных. База данных может располагаться на локальном компьютере, а может располагаться на удаленном сервере.

Перед началом работы необходимо создать новую базу данных для хранения портфеля проектов. Для этого необходимо в меню выбрать "Данные -> Локальная база данных -> Создать". При этом создается новая база данных sqlite3.

Формат базы данных:

    Таблицы
      Portfolio - таблица портфелей проектов

Описание таблиц

    Portfolio
      id
      name
